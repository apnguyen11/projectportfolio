React-Redux BattleShip

Capstone project consisting of a four member team. Primary purpose of this project was to learn to integrate React with Redux. A two player game was created using a firebase database to communicate between both players. Socket.io and node was used to create a chat to communicate between players and viewers. Saga was implemented to facilitate asyncronous actions for player state and attack.

Bank Hub

Member of a 3 person team creating a replica banking website. This project also used used node and express but a postgreSQL database was also implemented to save user information. This project taught us to use knex to create seeds and migrations to add users to a database. Additional fake users were generated by using FAKER. My main responsibilities included but were not limited too creating user authentication via Passport, integrating Travis CI, and form validation.

Check out the repository on Github!

Photography Portfolio

Individual solo project with the purpose of learning React. The portfolio focuses on using react components and using props to set information and pictures in bootstrap cards. Images were stored on firebase storage and the information for the bootsrap cards were stored and retrieved from a firebase database. A react lightbox was also created for the bootstrap cards to display a full-sized image
